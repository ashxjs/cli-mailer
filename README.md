# CLI-MAILER

## Install

### npm

```shell
    npx i
```

### yarn

```shell
    yarn
```

### Using

```shell
node build/main.js send <TO> --from <FROM> --template <SOME_ITEM>
```

| Argument   | Short | Long       | Description            |
| ---------- | ----- | ---------- | ---------------------- |
| `TO`       | n/a   | --to       | Destinary mail         |
| `FROM`     | n/a   | --from     | Verified email address |
| `TEMPLATE` | n/a   | --template | template items         |
